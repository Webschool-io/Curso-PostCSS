# Exercício do tema Variáveis

## O objetivo é otimizar o seguinte trecho de CSS com o conteúdo da aula

```scss
.nav {
  width: calc(6 * 120px);
  border-bottom: 1px solid #056ef0; 
}

.nav-item {
  color: black;
  padding-top: 30px;
  transition: color .3s ease-in-out; 
}

.nav-item:hover {
 color: #056ef0;
}

```
