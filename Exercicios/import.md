# Exercício do tema Import

### O objetivo é otimizar o seguinte trecho de CSS com o conteúdo da aula

```scss
.button {
 background-color: #26a69a;
 color: white;
 font-size: 14px;
 font-weight: 600;
}

.button:hover {
 opacity: .8;
}

.input {
 border: 1px solid #26a69a;
 color: black;
 font-size: 14px;
 font-weight: 600;
}

.input:focus {
 color: #26a69a;
}
```
